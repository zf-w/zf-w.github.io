一些关于游戏的想法

四子棋真是太适合去学习，去感受这种计算机解决问题的思路了，因为它特别容易去用计算机去高效地表.

四子棋的一些特性

第一点大概是首先合法操作和整体游戏 state 的简单表示要是计算机容易表示的，容易去查找的。比如说我感觉大多数古老的棋类游戏都有明确的格子和落点，或许可以说操作是比较有限的，并不涉及到小数。试想如果说围棋是以某种半径内的子数来计算吃子，而玩家可以下在任意 64 位的浮点数可以表示的地方，比如说 0.5 的位置而不一定是交叉点上，那不知道会是怎样一番情景。 首先大概这个游戏没有计算机就没法玩。不过或许也能蛮有意思。

第二点是要有明确的 terminal state 的判断，在 Connect Four 中，连成四个字就算赢，多么清楚。 像围棋，理论上的 terminal state 大概是每个地方不是子就是眼或者打劫，那不知道要下到什么时候。 对于人类玩家而言，一般来说是不会下到如此境地的。但是如果中盘结算，很多棋的死活便很难确定。或许一方玩家认为已经死了，但是实际还有活路，但是按照玩家的水平去玩又必不能救活，这便使输赢的判断变嘟很困难。

第三点是是否存在循环，循环的情况也会使情况变的十分复杂，或许谁先放弃循环就是决定胜负的一手，这就很难说谁胜谁负了. 四子棋就不存在这种情况.
