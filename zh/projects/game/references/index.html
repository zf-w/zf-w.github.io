货物手册

我读了第 2.5 节来学习项目结构的最佳实践. 我还读了第 3.1 节, 关于如何明确项目使用了哪些库, 去让我的代码更原子化. 听着不像人话...

Rust 实例

链接：https://doc.rust-lang.org/rust-by-example/generics/assoc_items/types.html

我读了第 14.8.2 节来了解如何使用"associated types"去更整洁的把不同部分的程序连接起来. 比如说, 我觉得确实"游戏自己定义自己的局面或者说状态"比这个游戏是一个关于某些<游戏状态>的游戏"更自然一些.

链接: https://github.com/emk/heroku-buildpack-rust

我学习了如何制作自定义构建包，特别是如何使用 Heroku 缓存目录制作它。
